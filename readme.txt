== Description ==

Test it Yourself is a wordpress puglin that will allow you to easily insert an editor to let the visitors test snippets codes.
For a demo and more info please click [HERE](http://blog.timersys.com/plugins/test-it-yourself-plugin-para-wordpress/).


== Installation ==


1. Upload `test-it-yourself` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use [testiy] shortcakes in your post

== Frequently Asked Questions ==

= How can i add more than one [testiy] shortcode? =

Just add new id`s like this: [testiy id="1"]
